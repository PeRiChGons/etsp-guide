# Recursos visuales

Las imágenes públicas de la guía se organizan según el [plan de recursos visuales](../../docs/PLAN-DE-RECURSOS-VISUALES.md).

## Estado de una imagen

- `pendiente`: todavía no existe un recurso válido.
- `extraído`: se ha separado de una captura o fotograma.
- `revisado`: nombre, forma y uso han sido comprobados.
- `publicado`: se utiliza en una página de la guía.

Los vídeos y capturas originales de investigación no se guardan aquí. Esta carpeta contiene únicamente recursos optimizados para la web y sus fichas de procedencia.

## Nombre de archivo

```text
tipo-nombre-variante-v01.ext
```

Ejemplos:

```text
icono-dark-phoenix-v01.png
retrato-dragon-lancer-v01.webp
ficha-sistema-spirit-v01.webp
objeto-basic-cleansing-pill-v01.webp
habilidad-sky-piercer-v01.webp
fondo-seccion-spirit-v01.webp
decoracion-marco-dorado-v01.png
```

No se deben introducir estadísticas, nombres o efectos dentro de una imagen si el dato debe poder editarse o traducirse.
