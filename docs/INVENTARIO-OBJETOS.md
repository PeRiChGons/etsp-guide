El vídeo `Outfix Spirit.mp4` se ha incorporado como muestra visual de Spiritfox; aparecen prendas de cabeza como `Golden Abyss Hairdress`, `Skyward Hairdress`, `Echoed Dream`, `Prosper`, `Distant Star Hairdress`, `Lantern Hairdress`, `Bright Moon Hairdress`, `Purple Fog Hairdress` y `Heaven's Match Hairdress`. El nombre y la ficha son comunes; la representación visual cambia según el personaje masculino o femenino. Los dos últimos citados ya tienen recorte propio del tooltip.

## Proceso de extracción

1. Localizar el fotograma donde el nombre y el icono aparecen juntos.
2. Guardar una copia completa como evidencia de investigación.
3. Recortar el icono sin texto ni interfaz para la ficha pública.
4. Añadir vídeo y marca de tiempo a `reference-data.js`.
5. Marcar la ficha como `confirmado` u `observado` según lo que permita leer la pantalla.
