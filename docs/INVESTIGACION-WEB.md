# Registro de investigación web

Última revisión: **14 de julio de 2026**.

Este documento registra qué información pública se ha localizado, qué contradicciones existen y qué datos todavía necesitan una comprobación directa dentro de Eternal Sword Pact. Su objetivo es evitar que una descripción repetida en Internet termine publicada como un hecho sin pruebas.

## Prioridad de las fuentes

1. Pantallas y textos del juego en la versión utilizada por el autor.
2. Tiendas y canales oficiales del desarrollador.
3. Vídeos donde la interfaz y los valores puedan verse con claridad.
4. Guías de terceros, usadas solo como orientación para nuevas comprobaciones.
5. Opiniones de jugadores, identificadas siempre como experiencias personales.

### Límite de las notas oficiales

Las fichas oficiales consultadas no incluyen un historial detallado de sistemas internos. No se ha encontrado en ellas una nota que explique Wisp, Spirit Root, Technique, SwordFlight o Zodiac. Por eso no se puede atribuir una fecha oficial de lanzamiento ni un funcionamiento completo a esos sistemas usando únicamente las tiendas.

Estas fuentes ayudan a localizar menús y temas, pero no sustituyen a una captura del juego. Las guías de BlueStacks se contradicen entre sí al describir el tipo de daño de Spirit Fox y Lunarborn. También existen listas de clases basadas en valoraciones subjetivas. Ninguna de esas afirmaciones se incorporará como recomendación definitiva sin una prueba directa.

La guía indonesia menciona ranuras de equipo, Copper, Mount Essence y mejoras de montura. Es un buen índice de temas para capturar, pero esos nombres, costes y límites no se consideran confirmados hasta comprobarlos en la versión del autor. El hilo de Reddit reúne experiencias de jugadores sobre tareas diarias, pases y gasto, no documentación oficial de mecánicas.

## Wisp: datos confirmados el 14 de julio de 2026

La captura aportada permite registrar sin interpretación:

- `Wisp Array Level: 7`;
- `Power: 22601915`;
- progreso `403/435`;
- doce posiciones ocupadas y cuatro vacías en esa pantalla;
- valores visibles de `Aptitude`: 330, 386, 373, 441, 445, 225, 470, 500, 600, 591, 514 y 450;
- pestañas `Sprite` y `Sacrifice`;
- botones `Quick Summon` y `Summon`;
- accesos `Spirit Beast Bag`, `Resonance`, `Stats Overview` y `Northern Abyss Order`;
- texto `When opened, Wisp Array Level EXP +1`;
- contador `9d 06h32m` junto a `Northern Abyss Order` en el momento de la captura.

El contador no demuestra por sí solo si Northern Abyss Order es un evento, una temporada o un desbloqueo. Tampoco se conocen todavía las estadísticas de cada Wisp, el cálculo de Aptitude, los costes, las probabilidades de invocación o el efecto de Sacrifice.

## Criterio editorial

- Un dato visible en una captura se marca como **confirmado en la captura**.
- Una explicación de una tienda oficial se marca como **fuente oficial**.
- Una afirmación de una guía o vídeo externo se marca como **fuente secundaria**.
- Una experiencia del autor se presenta como prueba personal, no como regla universal.
- Toda contradicción o ausencia de evidencia se mantiene como **Contenido pendiente**.
