# Lista de pendientes y niveles de evidencia

Este documento es la lista viva de trabajo de **Eternal Sword Pact Guía**. Sirve para distinguir lo que ya puede consultarse de lo que todavía necesita una captura, una repetición en el juego o una fuente oficial.

## Cómo interpretamos la información

Una afirmación puede tener uno de estos estados:

- **Confirmado por interfaz:** el nombre, botón, pestaña, valor o estado aparece claramente en una captura o pantalla del juego.
- **Observado en material:** aparece en un vídeo, PDF o fuente externa y se conserva como observación fechada, pero todavía no se generaliza a todas las cuentas o versiones.
- **Pendiente de verificar:** falta una pantalla, una descripción completa, una repetición o una fuente fiable antes de presentarlo como regla del juego.

La guía publica desde ahora los datos confirmados y observados con su contexto. Las partes pendientes permanecen visibles como tales para que se sepa exactamente qué falta.

## ¿Son los vídeos 100 % fiables?

No. Un vídeo es una fuente muy útil, pero no convierte automáticamente todo lo que aparece en una regla universal.

Sí es fiable para afirmar que, en ese momento y en esa cuenta, se veía una pantalla, un nombre, un botón, una secuencia de pasos o un valor concreto. No basta por sí solo para afirmar que:

- el coste es igual para todos los niveles;
- la recompensa o el reinicio siempre funcionan así;
- el sistema no ha cambiado desde la grabación;
- una opción es gratuita o P2W en todos los servidores;
- una recomendación de clase sirve para todas las versiones;
- una mecánica oculta funciona como parece en un vídeo editado.

Por eso los vídeos se usan como **evidencia observada**, junto con la fecha y el nombre del archivo. Para elevar un dato a confirmado buscamos una segunda captura, el icono de ayuda del juego, una repetición en otra cuenta o una fuente oficial.

## Cómo pasar un dato a confirmado

1. Capturar la pantalla completa y, si existe, el icono `?` o la descripción del juego.
2. Repetir el comportamiento en otra situación, nivel o cuenta cuando pueda variar.
3. Anotar versión, fecha y condiciones de la prueba.
4. Comparar con una fuente oficial o una segunda evidencia independiente.
5. Actualizar la página maestra y retirar el aviso de pendiente solo de ese dato concreto.

La lista se actualizará a medida que se incorporen nuevas capturas, vídeos o comprobaciones dentro del juego.
